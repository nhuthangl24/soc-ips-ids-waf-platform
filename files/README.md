# Tải file này về, sau đó làm theo các bước sau

- Vào **Edit** -> **Virtual Network Editor** -> **Change Settings** -> **Import**, sau đó chọn file mà mình vừa tải về.

> Chúc các bạn thành công !




# Download this file, then follow the steps below

- Go to **Edit** -> **Virtual Network Editor** -> **Change Settings** -> **Import**, then select the file you just downloaded.
